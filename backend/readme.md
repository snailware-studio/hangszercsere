# Hangszercsere.hu backend

## SQLite adatbázis
A `database.db` fileban tárol és olvas, ha nincsen akkor első futtatáskor készít egyet.

## Multer file tárolás
A `multer` npm csomag segítségével tárol fájlokat.

## Portok használata
A `3000` portot használja.

## Futtatás
```
npm install
npm start
```
